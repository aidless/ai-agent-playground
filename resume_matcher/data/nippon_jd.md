# AI Agent 开发工程师 — 立邦投资有限公司

## 公司背景

- **企业类型**: 外资（非欧美），大型企业（5000-10000 人）
- **行业**: 涂料行业全球巨头，行业地位稳定，抗风险能力极强
- **部门属性**: AI 团队属于前沿技术探索部门，预算充足，技术氛围好

## 岗位信息

- **工作地点**: 上海
- **招聘对象**: 2026 届应届生，本科及以上，无强制经验要求
- **核心定位**: AI 智能体 / 大模型应用开发工程师

## 核心工作内容

### 1. AI Agent 系统开发（核心）
- 基于大语言模型，设计开发具备"任务理解→任务拆解→工具调用→执行反馈"能力的智能体应用
- 负责 Agent 的技能（Skills）封装、工具调用机制开发、安全沙盒环境搭建、任务执行链路优化
- 围绕真实业务场景，推进智能体从原型验证到落地迭代的全流程

### 2. 工具与系统集成
- 对接业务系统接口、第三方服务接口，让 AI Agent 能调用外部工具（比如数据库、业务系统 API），提升执行能力
- 负责接口联调、异常排查、性能优化，保障系统稳定运行

### 3. 场景落地与效果优化
- 参与业务场景分析，验证 AI Agent 的可行性，推动项目落地
- 持续优化智能体的任务拆解准确率、执行效率、稳定性和用户体验，复盘运行问题，提升系统可维护性
